# Staff Attendance & Period Monitoring System

This project is a web application for managing staff attendance and period allocation in a college department. It includes a dashboard for HOD, admin panel for staff/timetable management, manual attendance marking, and attendance reports.

## Features
- HOD Dashboard: View daily timetable with color-coded attendance status.
- Admin Panel: Add/edit staff, set timetable allocations.
- Attendance Marking: Manual marking with real-time status updates.
- Reports: Generate daily/weekly attendance tables.

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js
- Database: JSON files (for simplicity)

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `npm start`.
4. Access the app in your browser at `http://localhost:3000`.

## Folder Structure
- `/public` - Static frontend files
- `/src` - Backend code (Node.js)
- `/data` - JSON files for staff, timetable, attendance
